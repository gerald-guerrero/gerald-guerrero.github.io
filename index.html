Projects

Gs-Renovations

Gs-Renovations is a website I built for a small family business focused on home renovations. It is hosted with Azure and was built using ASP.NET Core for the backend and React.JS for the frontend. the website is responsive, using media queries to handle different screen sizes, and includes four pages with mostly static content promoting the business along with a form feature using SMTP, so potential clients will be able to send requests to the company.

Hearth-Sensei

Hearth-Sensei is a website built using React, the Flask framework, and the Hearth Stone API. The website is meant to be used as a battle simulator for the various cards in the Hearth Stone game. I implemented the profile pages and visible battle history for the individual users. I also designed the implemented the functionality needed for deployment to Heroku.

View Project

Movie Rater

The movie rater is a website built using the Flask framework as well as the TMDB and Wikipedia API to display movie information. The two APIs listed above are used to pull information about the movie on the webpage and display that information along with the associated reviews left by the user. I implemented an account feature, so each individual user will have different customer reviews, and the user has the option to leave reviews on the movie listed on the screen.
